# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/797305a9-10f5-49b7-b2f3-9eee1d99743b

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `BEDROCK_API_KEY` in [.env.local](.env.local) to your Amazon Bedrock API key
3. Run the app:
   `npm run dev`
